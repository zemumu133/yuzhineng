# 宇智能本地项目

## v2 开源路线

项目后续统一命名为“宇智能”，定位为 AI 获客与内容运营客户端。

旧版本地控制台保留为历史版本、调试工具和阶段 1.x 参考，不再作为正式产品主线继续从零扩张。

v2 工作区位于：

`D:\OpenClaw\v2`

v2 采用“成熟开源项目 + 最小集成 + 国内获客 Skill/API/Workflow + 审批安全层”的路线，优先验证 LobsterAI、Sim.ai、搜索组件和 PocketBase 等候选组件。

高敏感能力默认关闭，包括自动发帖、自动评论、自动私信、自动邮件、批量导出和真实平台操作。后续如需开启，必须由用户手动开启，并配套风险提示、限流、审计日志和回滚方式。

所有后续实质开发任务必须先走 `v2/CODEX_WORKFLOW_CN.md`。任何新功能都必须先通过 `open_source_scout` 做开源复用判断，并通过 `PHASE_GATE_CHECKLIST_CN.md` 后才能提交。
