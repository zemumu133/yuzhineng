# 东莞制造业获客工作流

这是“宇智能”面向东莞本地制造业企业的第一版获客工作流模板。它把通用 Agent 能力包装成制造业老板和运营能看懂的路径：输入产品和工厂能力，输出客户方向、内容草稿、社媒计划、评论/私信草稿、账号养成计划和工厂销售交接单。

## 面向客户

- 包装厂
- 五金厂
- 健身器材厂
- 电子配件厂
- 塑胶厂
- 家居用品厂
- 外贸型工厂
- 内贸批发型工厂

## 当前边界

- 只生成草稿和待办。
- 不真实发帖。
- 不真实评论。
- 不真实私信。
- 不真实发邮件。
- 不登录社媒账号。
- 不抓取非公开数据。

## 普通用户路径

1. 选择“我要找客户”。
2. 填写工厂类型、产品、目标客户和平台。
3. 系统生成获客报告和内容草稿。
4. 用户审核后人工发布或人工跟进。
5. 把高价值客户交给工厂销售或老板。
