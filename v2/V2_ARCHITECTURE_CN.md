# v2 推荐架构

## 总体架构

v2 采用“开源底座 + 本地业务 Skill + 审批安全层”的架构。

## 组件分工

### LobsterAI

候选用途：桌面客户端壳、OpenClaw 本地运行入口、用户配置界面和打包发布验证。

验证重点：

- 是否方便嵌入本地 OpenClaw 控制台。
- 是否支持 Windows 用户安装、启动、升级和迁移。
- 是否适合非技术用户使用。

### Sim.ai

候选用途：Agent 工作流编排、可视化节点、任务执行状态和多 Agent 协作验证。

验证重点：

- 是否能表达市场、内容、销售、风控、数据等 Agent 分工。
- 是否能接入 DeepSeek 和本地 API。
- 是否能把执行结果回写到 OpenClaw 数据层。

### OpenClaw

用途：本地数据目录、脚本、审计、报告、审批队列、旧版本参考和 v2 运行胶水层。

### DeepSeek

用途：主模型。优先使用 DeepSeek V4 Pro 做中文任务规划、公开研究总结、内容生成和双审。

### domestic_signal_growth Skill

用途：封装国内获客、社媒内容、公开信息研究、潜客评分、审批草稿和复盘报告能力。

### Search Stack

候选组件：

- SearXNG：聚合搜索候选。
- Crawl4AI / Firecrawl：公开网页读取和结构化抽取候选。
- Meilisearch：本地索引、去重和检索候选。

### Membership

候选组件：PocketBase。用于会员、授权、设备绑定、轻量后台和本地/私有部署验证。

## 安全策略

高敏感功能默认关闭，包括邮件发送、私信、评论、发帖、删除内容、批量导出客户数据和付费推广。

后续如需开启，必须满足：

- 明确风险提示。
- 用户手动确认。
- 审批记录。
- 审计日志。
- 可回滚配置。
