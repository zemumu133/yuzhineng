# 阶段门禁清单

任何阶段进入下一步前，必须满足以下门禁。

## 开源复用门禁

- 已检查成熟开源项目
- 已检查官方 SDK/API
- 已检查插件/Webhook/CLI/MCP/Skill 接入可能性
- 已说明为什么不自研或为什么必须最小自研

## 最小改动门禁

- 已列出修改文件
- 已说明不修改范围
- 没有大规模重构
- 没有重复造轮子
- 没有把 PoC 写成正式大系统

## 安全门禁

- 没有提交密钥
- 没有读取 secrets 内容
- 没有提交 Cookie、Token、浏览器 Profile
- 没有真实发送邮件、私信、评论、发帖
- 高敏感功能默认关闭
- 高敏感功能允许用户后续手动开启
- 有审计日志设计或说明

## 易用性门禁

- 普通用户能理解入口
- 不暴露过多技术概念
- 有明确下一步
- 失败状态能看懂
- 不堆按钮
- 不堆表格

## 测试门禁

- 已运行相关测试
- 已记录失败原因
- 没有伪造成功
- Git 工作区状态已检查
- 需要提交的已提交，不该提交的已忽略

## 报告门禁

- 中文报告已更新
- 结论明确：继续 / 继续但有风险 / 暂停 / 放弃
- 下一步明确
- 风险明确
