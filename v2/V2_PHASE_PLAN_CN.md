# v2 阶段计划

## R0：项目路线重置与 v2 工作区初始化

目标：盘点现状、保护旧项目、创建 v2 隔离目录和路线文档。

## Phase 1A：LobsterAI PoC

目标：验证 LobsterAI 是否适合作为 Windows 桌面客户端壳和 OpenClaw 打包入口。

## Phase 1B：Sim.ai PoC

目标：验证 Sim.ai 是否适合作为 Agent 工作流编排、可视化和执行状态底座。

## Phase 2：domestic_signal_growth Skill

目标：把国内获客、公开研究、内容生成、审批草稿和报告能力封装为可移植 Skill/API/Workflow。

## Phase 3：Search Stack PoC

目标：验证 SearXNG、Crawl4AI/Firecrawl、Meilisearch 的公开搜索、网页读取和本地索引能力。

## Phase 4：Membership PoC

目标：验证 PocketBase 或类似方案的会员、授权、设备绑定和轻量后台能力。

## Phase 5：客户端集成

目标：把客户端壳、工作流引擎、业务 Skill、搜索和会员能力串成可用桌面产品。

## Phase 6：小范围真实客户试用

目标：面向少量真实用户进行人工监督试用，验证产品价值、风险边界和商业可行性。
