# 宇智能 v2

宇智能是面向中小企业、商家和运营团队的 AI 获客与内容运营客户端。

本目录是 OpenClaw 项目的 v2 开源路线工作区。

旧版 `D:\OpenClaw` 本地控制台保留为历史版本、调试工具和阶段 1.x 参考，不再作为正式产品主线继续从零扩展。

v2 的方向是复用成熟开源项目：

- LobsterAI：桌面客户端壳、OpenClaw 打包与用户侧交互验证。
- Sim.ai：Agent 工作流编排、可视化和运行验证。
- OpenClaw：本地数据、脚本、审计、安全规则和已有阶段资产。
- DeepSeek：主模型，优先用于中文增长任务规划、研究、内容和审核。
- domestic_signal_growth Skill：国内获客、内容运营和公开信息研究业务能力。
- Search Stack：公开搜索、网页读取、索引和来源追踪。
- PocketBase：会员、授权、设备绑定和轻量后台能力候选。

v2 不默认开启邮件、私信、评论、发帖等真实外部动作。高敏感能力必须默认关闭，经过风险提示、人工确认和审计日志后才允许后续阶段手动启用。
